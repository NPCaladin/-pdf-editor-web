# GitHub Desktop 저장소 경로 수정하기

현재 "PDF-editor" 저장소가 다른 폴더에 생성되었습니다.
현재 작업 폴더(`C:\Users\master\Desktop\new_aladin`)를 저장소로 설정해야 합니다.

---

## 해결 방법

### 방법 1: 기존 저장소 제거 후 새로 생성 (추천)

1. **GitHub Desktop에서**
   - File → Options → Repositories 클릭
   - "PDF-editor" 저장소 선택
   - "Remove" 클릭
   - "Keep" 선택 (파일은 유지)

2. **새 저장소 생성**
   - File → New Repository 클릭
   - **Name**: `pdf-editor-web`
   - **Local Path**: `C:\Users\master\Desktop\new_aladin`
     - **중요**: "Choose..." 버튼을 클릭하여 정확히 이 폴더 선택!
   - Git ignore: None
   - "Create repository" 클릭

3. **확인**
   - 이제 "Changes" 탭에서 파일들이 보여야 합니다:
     - ✅ `app.py`
     - ✅ `requirements-web.txt`
     - ✅ `Procfile`
     - ✅ `runtime.txt`
     - ✅ `templates/` 폴더
     - ✅ `static/` 폴더

---

### 방법 2: 현재 폴더를 Git 저장소로 추가

1. **GitHub Desktop에서**
   - File → Add Local Repository 클릭

2. **폴더 선택**
   - "Choose..." 클릭
   - `C:\Users\master\Desktop\new_aladin` 선택
   - "Add repository" 클릭

3. **확인**
   - 파일들이 "Changes" 탭에 나타나야 합니다

---

## 중요: 경로 확인

저장소를 만들 때 반드시:
- `C:\Users\master\Desktop\new_aladin` 폴더를 선택해야 합니다
- "Choose..." 버튼을 사용하여 정확한 경로 선택
- 다른 폴더(예: `PDF-editor`)가 아닌지 확인
