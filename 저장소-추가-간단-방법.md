# GitHub Desktop 저장소 추가 (간단한 방법)

Options에 "Repositories" 메뉴가 없어도 괜찮습니다!
다른 방법으로 저장소를 추가할 수 있습니다.

---

## 방법 1: File 메뉴에서 직접 추가 (가장 쉬움)

### 1단계: 현재 폴더를 Git 저장소로 만들기

1. **PowerShell 또는 명령 프롬프트 열기**
   - Windows 키 + R
   - `powershell` 입력
   - Enter

2. **현재 폴더로 이동**
   ```powershell
   cd C:\Users\master\Desktop\new_aladin
   ```

3. **Git 초기화** (Git이 설치되어 있다면)
   ```powershell
   git init
   ```
   
   **만약 Git이 설치되어 있지 않다면:**
   - 이 단계는 건너뛰고 다음 단계로 진행

### 2단계: GitHub Desktop에서 추가

1. **GitHub Desktop 열기**

2. **File 메뉴 클릭**
   - File → Add Local Repository 클릭

3. **폴더 선택**
   - "Choose..." 버튼 클릭
   - `C:\Users\master\Desktop\new_aladin` 폴더 선택
   - "Add repository" 클릭

4. **확인**
   - 이제 "Changes" 탭에서 파일들이 보여야 합니다!

---

## 방법 2: 새 저장소 생성 (Git 초기화 없이)

1. **GitHub Desktop에서**
   - File → New Repository 클릭

2. **저장소 설정**
   - **Name**: `pdf-editor-web`
   - **Local Path**: `C:\Users\master\Desktop\new_aladin`
     - **중요**: "Choose..." 버튼을 클릭하여 정확히 이 폴더 선택!
   - Git ignore: None
   - **"Create repository"** 클릭

3. **확인**
   - "Changes" 탭에서 파일들이 보여야 합니다

---

## 방법 3: 드래그 앤 드롭 (일부 버전에서 가능)

1. **파일 탐색기 열기**
   - `C:\Users\master\Desktop\new_aladin` 폴더 열기

2. **GitHub Desktop에 드래그**
   - 폴더를 GitHub Desktop 창으로 드래그 앤 드롭
   - 자동으로 저장소로 추가됨

## 문제 해결

### 파일이 여전히 안 보임

1. **폴더 경로 확인**
   - 파일 탐색기에서 `C:\Users\master\Desktop\new_aladin` 열기
   - 파일들이 실제로 있는지 확인

2. **GitHub Desktop 새로고침**
   - F5 키 누르기
   - 또는 File → Refresh

3. **저장소 다시 추가**
   - File → Add Local Repository
   - 정확한 경로 선택
